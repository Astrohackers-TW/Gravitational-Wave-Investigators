# 重力波審查員 (Gravitational Wave Investigators)

## 使命
[可重複性](https://en.wikipedia.org/wiki/Reproducibility)是科學方法中一項重要的原則。你們的使命是與其他重力波審查員合力審查並重現[LIGO開放科學中心](https://losc.ligo.org)所提供的LIGO觀測資料分析步驟，找出LIGO研究團隊是如何發現重力波
